# 名城大学-亀谷研究室-ヒートマップ閲覧システム

## 起動のための手順1 : npm installを実行する
```
npm install
```

## 起動のための手順2 : npm run serve で開発用web サーバを起動する
```
npm run serve
```

## 起動のための手順3: http://localhost:8080 へWeb ブラウザでアクセス
